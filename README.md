# 보컬 MBTI 진단 서비스

나만의 시그니처 보컬 MBTI를 알아보는 웹 서비스입니다.

## 기능

- 음성 녹음을 통한 보컬 MBTI 진단
- 16가지 MBTI 유형별 상세 분석
- 맞춤형 트레이너 추천
- 보컬 밸런스 레이더 차트
- 결과 공유 기능

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
- Web Audio API
- SVG
